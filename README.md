# PharmaPulse 💊

PharmaPulse is a comprehensive, full-stack pharmacy management system designed to optimize workflow efficiency in modern healthcare environments. It provides real-time tracking for patient queues, inventory stock levels, doctor appointments, and system metrics through an intuitive dark neon-themed dashboard.

## 🚀 Key Features
* **User Authentication**: Secure login and registration portals for staff and administrators.
* **Dynamic Dashboard**: Real-time metrics overview including total orders, low-stock warnings, and dispatched queues.
* **Live Patient Queue**: Live tracking of patients, medication requirements, statuses (Pending/Dispensed), and timestamps.
* **Inventory Management**: Automated tracking of medicine quantities, expiry dates, and threshold-based low-stock alerts.
* **Doctor Appointments**: Seamless management of patient-doctor appointments and scheduling statuses.

## 🛠️ Tech Stack
* **Frontend**: React.js (Vite), React Router, Axios, Custom CSS (Neon-box theme)
* **Backend**: Node.js, Express.js
* **Database**: MongoDB & Mongoose (with automatic timestamps)
